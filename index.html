Scratch

The Impossible Game

8/30-9/5

Our Scratch project, designed and modeled after the popular browser-game "World's Hardest Game", was created to be as satisfying when finally completed as the original game. Each level was designed using the program "Paint.NET", and every obstacle and monster was individually programmed based on it's position on the map. We were originally worried about using the "touching color" element, however, we encountered almost no problems. The only encumbrance we faced was on level 3, when the player often had difficulty moving through the spawn-point. A simple redesign of the spawn-point fixed it. I believe this project specifically helped with brainstorming for creative ideas. My partner and I were given experience into what it's like to throw out and criticise ideas, and which ideas we could implement into the game - with the skills we had and time given. This experience has helped with every project sense.
